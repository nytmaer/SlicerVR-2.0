# SlicerVR 2.0

**A web-based MP4 video trimmer designed for Meta Quest 3 and desktop browsers. Trim your VR gameplay recordings without removing your headset.**

## Features

- ✂️ **Visual Trim Window** - Drag handles to set start/end points
- 🥽 **Quest 3 Optimized** - Works in VR browser with controller-friendly UI
- 📱 **Cross-Platform** - Desktop, mobile, and VR headsets
- 💾 **Privacy First** - Video never leaves your device
- 🚀 **Zero Setup** - Single HTML file, no dependencies

## How to Use

1. **Upload** - Drag & drop MP4 file
2. **Trim** - Drag handles to select clip
3. **Export** - Download trimmed MP4

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play/Pause |
| `[` `]` | Jump to start/end |
| `←` `→` | Scrub timeline |
